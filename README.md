# Yev Steam Deposit Calculator & CS2 Arbitrage Tool

A calculator for CS2 investors and traders, built with Python (Streamlit). It helps accurately calculate the profit of topping up your Steam balance via third-party marketplaces and identifies the most profitable entry points for buying skins.

## Key Features
- **Profit Calculator:** Precise profit calculation accounting for all third-party marketplace fees and Steam Community Market taxes.
- **Steam Integer Math:** Native support for Steam's specific rounding logic for integer-only currencies (e.g., UAH/₴), preventing "impossible price" calculation errors.
- **Advanced Cross-Rates:** Advanced multi-currency mode supporting simultaneous conversions across different currencies (e.g., Card -> Site -> Steam).
- **Fee Templates:** Built-in fee presets for popular third-party marketplaces (CSFloat, SkinSwap, etc.) with dynamic updates.
- **Quantity Support:** Automatic fee scaling and profit calculation for bulk item purchases.
